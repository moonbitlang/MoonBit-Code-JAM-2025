## 构建与运行（WASM）

1) 构建
```
moon build --target wasm --release
```
2) 复制产物
```
# PowerShell
Copy-Item target/wasm/release/build/main/main.wasm artifact/game.wasm -Force

# Bash
cp target/wasm/release/build/main/main.wasm artifact/game.wasm
```
3) 本地运行
```
cd artifact
python -m http.server 8000
```
访问：`http://localhost:8000`
