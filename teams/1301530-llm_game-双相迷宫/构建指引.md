## 编译（MoonBit）
1) 进入源码目录
```
cd source_code/
```
2) 编译 wasm-gc 目标
```
moon build --target wasm-gc
```
3) 将编译产物复制到作品目录
```
copy source_code/target/wasm-gc/release/build/lib/lib.wasm "MoonBit-Code-JAM-2025-main/teams/1301530-llm_game-双相迷宫/artifact/game.wasm"
```

## 构建与产物说明（源码 vs 编译物）
- 源码（修改逻辑时编辑）
  - `source_code/lib/lib.mbt`：游戏核心（状态、关卡、导出接口）
  - `source_code/lib/moon.pkg.json`：导出函数清单
  - `source_code/moon.mod.json`：模块元信息
- 编译产物（自动生成）
  - `source_code/target/wasm-gc/release/build/lib/lib.wasm`：WASM 文件
  - `source_code/target/**`：中间产物（可删除后重建）
- 参赛产物（交付）
  - `teams/1301530-llm_game-双相迷宫/artifact/index.html`
  - `teams/1301530-llm_game-双相迷宫/artifact/game.wasm`
