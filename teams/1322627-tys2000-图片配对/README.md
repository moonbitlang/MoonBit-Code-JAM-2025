### 游戏简介：  
Pair Matching 游戏  
《图片配对》是一款经典的记忆配对解谜游戏，考验玩家的观察力与短期记忆力。游戏棋盘上布满背面朝上的卡牌，每张卡牌对应一个水果图案，相同图案的卡牌成对出现。玩家需通过翻牌尝试找到所有匹配对，成功配对全部卡牌即获胜。

视频介绍：https://www.bilibili.com/video/BV1j3CKBHEJf/

### 操作说明  
- 点击任意一张背面朝上的卡牌将其翻开。  
- 每次最多可翻开两张卡牌：若图案相同，则配对成功并保持正面朝上；若不同，则两张卡牌自动翻回背面。  
- 游戏开始前有 5 秒记忆时间供玩家预览所有卡牌位置。  
- 目标是在最短时间内完成所有配对。  
- 点击右上角声音按钮可开启/关闭音效。

### 技术特色：  
游戏状态（如卡牌位置、翻牌状态、匹配结果等）均由 WASM 模块管理，前端通过 Canvas 实现流畅的 2D 渲染与交互，支持自适应布局与音效反馈，兼顾性能与用户体验。
