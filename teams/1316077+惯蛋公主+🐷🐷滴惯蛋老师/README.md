## 游戏简介

《掼蛋小课堂》是“惯蛋公主”团队创作的一款掼蛋练习 DEMO。  
我们把发牌、座位、人格标签等信息做成场景化体验，让玩家在轻松的氛围里练习牌技，熟悉不同搭档组合带来的出牌节奏。  
设计理念强调“轻量、可视、可交互”，希望让掼蛋爱好者快速进入状态、反复实战演练。

**演示视频**：https://www.bilibili.com/video/BV1kHCTBYEgu/

## 操作说明

1. 打开页面后点击“开始游戏”，完成三个简单问题生成个人牌风。  
2. 在“座位设定”界面为队友与上下家选择人设后，点击“进入牌桌”。  
3. 牌桌加载后会自动洗牌并给四位玩家各发 27 张牌；  
   - 屏幕四角分别展示自己（左下）、队友（右上）、上家（左上）、下家（右下）的形象；  
   - 中央提示区用来显示刚刚打出的牌；  
   - 下方一排是“自己”的手牌。  
4. 将鼠标悬停在手牌上会有抬起效果，点击即可将该张牌打到桌面中央。重复点击不同的牌可以模拟出牌流程。  
5. 若想重新体验，点击“回到首页”重新测评并发牌。

## 技术特色

- **MoonBit × Web 前端联合驱动**：  
  - 通过 `window.moonbitGuandan.deal()` 接口封装发牌逻辑，可直接接入 MoonBit 输出的牌面数据；  
  - 手牌、牌堆、出牌区等状态集中管理，方便扩展。

- **可视化拼桌与手牌交互**：  
  - 四个座位使用绝对定位与人物卡片样式，明确“自己/队友/上下家”的相对位置；  
  - 中央出牌区自动滚动显示最新牌，便于观察。

- **发牌逻辑模块化**：  
  - 双副牌 108 张的构建、洗牌、排序完全独立封装；  
  - 结构为：生成牌堆 → 轮流发牌 → 排序 → 更新 UI，与真实掼蛋流程一致。
