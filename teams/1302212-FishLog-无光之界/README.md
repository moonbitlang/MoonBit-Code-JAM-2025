## 游戏简介

在这个世界中，古老的神祇早已堕落，文明的余烬在寒风中摇摇欲坠。你将扮演一个被诅咒的“无光者”，一个背负着沉重宿命的灵魂。为了寻找一丝渺茫的希望，亦或是为了一个复仇的执念，你必须踏入这片破碎的土地。

你将面对的，是拥有强大力量、被黑暗扭曲的敌人，是潜藏在历史阴影中的巨大秘密。每一次战斗都是一次生死的考验，每一次倒下都让你离疯狂更近一步。

准备好你的剑，管理好你的精力，在这片没有希望的土地上，挣扎求生。

## 操作说明
*  **键盘与鼠标 (Keyboard & Mouse):**
    * ` A / D`: 角色移动
    * ` J `: 轻攻击
    * `空格键 (Space)`: 跳跃
    * `F` : 互动
    * `E` : 打开物品栏 / 菜单
    * `K` ：冲刺
    * `Y` / `U` / `I` / `O` / `L` / `M` / `N` ：技能


## 技术特色

* **核心开发:** 游戏使用 **MoonBit** 语言作为主要开发工具，构建了高性能、可维护的游戏逻辑。
* **AI 设计:** 敌人 AI 具有挑战性，包含追击、攻击组合。
* **黑暗美术风格:** 游戏采用2D像素风格，通过压抑的色调、光影对比和细节丰富的环境设计，营造出一个破碎而残酷的黑暗世界。
