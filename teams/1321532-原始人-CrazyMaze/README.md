# Crazy Maze

## 游戏简介

简单的走迷宫小游戏，假装你是一个原始人，展开疯狂的迷宫游戏吧！

介绍视频：https://www.bilibili.com/video/BV1xp2uBHEPj

## 操作说明

按箭头键←↑↓→操作角色行动，达到右下角🏆处获得胜利；胜利后，按空格键SPACE刷新迷宫地图，可重新开始游戏。

## 技术特色

使用Prim算法和MoonBit提供的Selene游戏框架搭建迷宫场景。
