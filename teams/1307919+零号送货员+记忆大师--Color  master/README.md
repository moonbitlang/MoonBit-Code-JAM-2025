# 记忆大师 -- Color Master v2.0.0 版本（正式服）

## 游戏简介

**记忆大师 -- Color Master** 是一款由 **零号送货员** 开发的考验记忆力、颜色识别能力与文化素养的综合性游戏。游戏旨在通过多层次的颜色记忆挑战、传统文化知识与人文互动，提升玩家的综合认知能力，同时提供兼具教育意义与娱乐性的独特体验。

---

## 操作说明

### **通用操作**
1. **启动游戏**：点击"启动游戏"按钮进入加载界面
2. **选择模式**：在主菜单点击对应模式卡片（支持翻转效果）
3. **更换主题**：点击"更换主题"按钮随机切换渐变背景
4. **操作说明**：点击"操作说明"跳转b站查看游戏教程
5. **开发过程**：点击"开发过程"查看游戏开发文档

### **各模式专属规则**

#### **简单模式**
- **记忆时间**：3秒
- **干扰项**：2个，颜色差异度低
- **操作**：使用方向键选择，Enter确认

#### **困难模式**
- **记忆时间**：4秒
- **干扰项**：2个，颜色高度相似（ΔE < 5）
- **动态干扰**：选项位置随机浮动，增加辨识难度

#### **地狱模式**
- **记忆时间**：6秒
- **五行机制**：
  - 每道关卡都会有不同的五行颜色以及相生相克的关系
  - 不仅考验玩家的记忆能力，还会考验玩家的五行相生相克的理解

#### **古人模式**
1. **诗意寻色**：系统随机给出含颜色字的诗句，如"露染宫** **菊浅"（隐藏"黄"字）
2. **诗词互动**：
   - 在聊天框输入诗句片段，系统会补全并提示关键字
   - 内置关键词触发："你好"、"游戏"、"颜色"、"记忆"、"难"、"谢谢"、"再见"、"露染宫庭菊浅"等
   - 响应示例：输入"庭菊浅" → 系统回复："可是'露染宫**黄**菊浅'？黄字不可少"
3. **操作方式**：
   - 上方显示诗句与空缺色块
   - 下方12色盘选择正确答案
   - 方向键移动，Enter确认

## 技术特色

- **双语言架构**：Moonbit编译核心算法，JavaScript处理UI交互，性能与灵活性兼得
- **色彩科学**：生成干扰项，确保色差精准可控
- **五行系统**：内置五行生克关系，动态调整干扰项
- **诗词数据库**：收录经典诗句，支持模糊匹配与智能提示
- **干扰项生成**：12色相微移 + 饱和度/亮度双策略，保证每组12色无重复

## 设计理念
- **多维挑战**：从基础记忆到文化理解，层层递进，全方位锻炼大脑
- **科学融合**：创新性地将中国传统五行学说融入游戏机制，寓教于乐
- **文化传承**：通过诗词鉴赏与古人对话，在游戏中感受中华文化魅力
- **人文互动**：虚拟智者对话系统提供文化引导，打造沉浸式学习体验
- **视觉美学**：每套配色方案均经过精心调配，兼具美感与辨识度

**游戏模式：**
- **简单模式**：基础颜色记忆训练，适合入门玩家
- **困难模式**：高相似度色彩干扰，挑战辨识极限
- **地狱模式**：融合五行相生相克机制，错误选项会触发循环干扰
- **古人模式**：独创"诗词寻色"玩法，与虚拟智者对话解谜，体验文人雅趣
